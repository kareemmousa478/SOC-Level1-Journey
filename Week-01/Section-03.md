# Section 03 - Core SOC Solutions

## Overview

This section introduced the core technologies used by Security Operations Centers (SOCs) to detect, investigate, and respond to cyber threats.

---

## Topics Covered

- Endpoint Detection and Response (EDR)
- Security Information and Event Management (SIEM)
- Splunk Basics
- Elastic Stack Basics
- Security Orchestration, Automation and Response (SOAR)

---

## Key Concepts

### SIEM

A centralized platform that collects, correlates, and analyzes logs from multiple systems to detect suspicious activities.

### EDR

A security solution installed on endpoints that continuously monitors endpoint activities, detects threats, and supports investigation and response.

### Splunk

A SIEM platform used to collect, search, analyze, and visualize machine-generated data.

### Elastic Stack

A collection of tools used for log collection, storage, searching, and visualization.

### SOAR

A platform that automates repetitive security tasks and helps analysts respond faster to incidents.
